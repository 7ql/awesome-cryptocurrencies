# Nano

- orig. Raiblocks (launched Oct, 2015, rebranded in Jan, 2018)
- asynchronous p2p payments w/o fees, w/o expensive computing
- novel, block-lattice data structure:
- each user has its own blockchain and processing power
- users' blockchain keeps track of tx and balance history
- tx is then broadcast to the entire network and recorded in historical ledgers
- avg tx fee: $.002 (amount of electricity needed for pow to  process)
- https://nano.org/
